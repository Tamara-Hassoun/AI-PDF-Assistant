# AI PDF Assistant

Python AI tool to analyze PDF documents and answer questions based on the content. Converts text into embeddings and allows interactive querying via Streamlit interface.

# Features

- Extract text from PDF files
- Convert text to embeddings for AI processing
- Store data in vector database
- Simple interactive interface with Streamlit

# Technologies

- Python
- Streamlit
- NLP (Natural Language Processing)
- Vector Databases / ChromaDB

# How to Run

1. Open in Google Colab or local Python environment
2. Install required libraries (`pip install -r requirements.txt`)
3. Run the app (`streamlit run app.py`)
4. Upload PDF files and ask questions
